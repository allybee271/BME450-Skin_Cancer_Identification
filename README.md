# "Skin Cancer Identification Project” 
## Team members: Alexis Hawkins (Allybee271), Cailyn Bowerman (cailynbowerman) 
## Project decription Max ½ page. Include some information on: - what dataset you plan to use - goal of the project. 
## Skin cancer is a serious global public health crisis with an ever-increasing incidence and early detection significantly improves survival rates. A simple and reliable means of identifying melanoma would allow for earlier detection and peace of mind for patients. The purpose of this project is to distinguish dermoscopic photos of melanoma from other skin lesions. We plan to use a dataset from ISIC Challenge which offers thousands of public high-quality human-validated training and testing images of various malignant and non-malignant skin lesions. The architecture of the model will be modified to analyze the train and test results of the different configurations.
