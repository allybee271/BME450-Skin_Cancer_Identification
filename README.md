# "Skin Cancer Identification Project” 
## Team members: Alexis Hawkins (Allybee271), Cailyn Bowerman (GitHubUserB) 
## Project decription Max ½ page. Include some information on: - what dataset you plan to use - goal of the project. 
## Skin cancer is a serious global public health crisis with an ever-increasing incidence and early detection significantly improves survival rates. A simple and reliable means of identifying melenoma would allow for earlier detection or peace of mind for patients. The purpose of this project is to distinguish dermoscopic photos of melanoma from other skin lesions. We plan to use a dataset from ISIC Challenge which offers public high-quality human-validated training and test sets of thousands of images of various skin lesions.
